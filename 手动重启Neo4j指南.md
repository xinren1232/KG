## 🔄 手动重启方法

### 方法1: 通过任务管理器（推荐）

1. **打开任务管理器**
   - 按 `Ctrl + Shift + Esc`
   - 或右键任务栏 → 选择"任务管理器"

2. **查找Neo4j进程**
   - 点击"详细信息"选项卡
   - 查找 `java.exe` 进程
   - 查看"命令行"列，找到包含 `neo4j` 的进程

3. **结束Neo4j进程**
   - 右键选择Neo4j的java.exe进程
   - 点击"结束任务"

4. **重新启动Neo4j**
   - 通过您原来启动Neo4j的方式重新启动
   - 或通过Neo4j Desktop重新启动

### 方法2: 通过服务管理器

1. **打开服务管理器**
   - 按 `Win + R`
   - 输入 `services.msc`
   - 按回车

2. **查找Neo4j服务**
   - 在服务列表中查找包含"Neo4j"的服务
   - 可能的服务名: `Neo4j`、`Neo4j-CE`、`Neo4j Community`

3. **重启服务**
   - 右键Neo4j服务
   - 选择"重新启动"

### 方法3: 通过Neo4j Desktop

1. **打开Neo4j Desktop**
2. **停止数据库**
   - 找到您的数据库实例
   - 点击"Stop"按钮
3. **启动数据库**
   - 点击"Start"按钮

### 方法4: 通过命令行（如果Neo4j在PATH中）

```bash
# 停止Neo4j
neo4j stop

# 启动Neo4j  
neo4j start
```
