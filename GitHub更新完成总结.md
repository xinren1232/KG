│   前端服务   │◄──►│   API服务   │◄──►│  数据库服务  │

- **前端**: Vue.js 3 + Element Plus + Vite + ECharts
- **后端**: FastAPI 2.0 + Python 3.x + Neo4j 5.23
- **部署**: Docker Compose + Nginx + systemd
