## 🎯 系统概述

质量知识图谱助手是专为手机研发质量部门设计的智能化管理平台，通过知识图谱技术整合测试流程、异常分析、历史案例等信息，提供智能问答和决策支持。

### 核心功能

- **📋 测试流程查询**: 根据产品和组件快速查找相关测试用例
- **🔍 异常指导分析**: 智能分析异常症状，提供解决方案
- **🕸️ 知识图谱可视化**: 交互式探索产品、组件、测试用例间的关系
- **🤖 智能问答**: 基于知识图谱的自然语言问答
- **📊 数据统计监控**: 实时系统状态和数据统计

## 📖 功能详解

### 1. 📋 测试流程查询

#### 功能说明
根据产品和组件查询相关测试用例，支持多维度筛选和详情查看。

#### 操作步骤
1. **选择产品**: 从下拉列表中选择目标产品
2. **选择组件**: 根据产品自动加载相关组件列表
3. **设置筛选条件**:
   - 优先级: 高/中/低/紧急
   - 类型: 功能测试/性能测试/兼容性测试等
4. **查看结果**: 点击测试用例查看详细信息

#### 测试用例详情包含
- 用例ID和标题
- 详细描述
- 测试步骤
- 期望结果
- 优先级和类型
- 关联组件

### 2. 🔍 异常指导分析

#### 功能说明
输入异常症状描述，系统智能匹配相关异常案例，提供完整的分析路径。

#### 操作步骤
1. **输入症状**: 在搜索框中描述遇到的异常症状
   - 示例: "摄像头无法对焦"
   - 示例: "手机发热严重"
2. **查看匹配结果**: 系统显示相似异常列表
3. **分析因果路径**: 查看 症状→根因→对策 的完整路径
4. **应用解决方案**: 根据推荐的对策执行修复

#### 异常信息包含
- 异常ID和标题
- 详细描述
- 严重程度
- 状态信息
- 症状列表
- 根因分析
- 解决对策
- 相似度评分

### 3. 🕸️ 知识图谱可视化

#### 功能说明
以图形化方式展示产品、组件、测试用例、异常等实体间的关系网络。

#### 操作步骤
1. **加载图谱**: 系统自动加载知识图谱数据
2. **交互操作**:
   - **拖拽**: 移动节点位置
   - **缩放**: 鼠标滚轮缩放视图
   - **点击**: 选择节点查看详情
3. **布局切换**: 选择不同的图谱布局算法
   - 网格布局
   - 圆形布局
   - 层次布局
   - 力导向布局
4. **筛选显示**: 根据节点类型筛选显示内容

#### 图谱元素说明
- **蓝色节点**: 产品
- **绿色节点**: 组件
- **橙色节点**: 测试用例
- **红色节点**: 异常
- **连线**: 表示实体间的关系

### 4. 🤖 智能问答

#### 功能说明
基于知识图谱的自然语言问答系统，支持复杂查询和智能推荐。

#### 操作步骤
1. **选择问题模板**（可选）:
   - iPhone 15摄像头测试流程
   - 电池异常分析
   - 屏幕测试要点
   - 常见故障排查
2. **输入问题**: 在对话框中输入自然语言问题
   - 示例: "iPhone 15的摄像头测试流程是什么？"
   - 示例: "手机电池发热的原因有哪些？"
3. **查看回答**: 系统提供详细回答，包括：
   - 主要答案内容
   - 置信度评分
   - 信息来源
   - 相关建议
4. **相似异常搜索**: 输入症状描述查找相似异常案例

#### 问答特性
- **上下文理解**: 支持多轮对话
- **智能推荐**: 提供相关问题建议
- **来源追溯**: 显示答案的知识来源
- **置信度**: 评估答案的可信程度

### 5. 📊 数据统计监控

#### 系统状态监控
- **API服务状态**: 实时监控API服务健康状况
- **数据库连接**: 检查Neo4j数据库连接状态
- **响应时间**: 监控系统响应性能

#### 数据统计
- **产品数量**: 系统中管理的产品总数
- **测试用例**: 测试用例总数和分类统计
- **异常记录**: 异常数据统计和趋势分析
- **知识节点**: 知识图谱中的节点和关系统计

## ❓ 常见问题

### Q1: 如何添加新的测试用例？
**A**: 目前系统支持通过Excel文件批量导入测试用例。请联系系统管理员获取模板文件和导入流程。

### Q2: 智能问答的准确性如何？
**A**: 系统基于知识图谱提供答案，准确性取决于数据质量。每个回答都会显示置信度评分，建议结合实际情况判断。

### Q3: 如何查看历史查询记录？
**A**: 智能问答模块会保存对话历史，可以在对话界面查看。其他模块的查询历史可通过浏览器历史记录查看。

### Q4: 系统支持哪些浏览器？
**A**: 推荐使用Chrome、Firefox、Safari、Edge等现代浏览器的最新版本。

### Q5: 如何导出查询结果？
**A**: 目前支持复制文本内容，后续版本将支持Excel和PDF导出功能。

## 🔧 故障排除

### 页面加载缓慢
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试刷新页面
4. 联系技术支持

### 搜索结果为空
1. 检查搜索关键词拼写
2. 尝试使用更通用的关键词
3. 确认数据已正确导入系统
4. 联系数据管理员

### 图谱显示异常
1. 尝试切换不同的布局算法
2. 检查浏览器是否支持WebGL
3. 降低显示的节点数量
4. 刷新页面重新加载

### 智能问答无响应
1. 检查问题描述是否清晰
2. 尝试使用问题模板
3. 确认API服务状态正常
4. 联系技术支持
