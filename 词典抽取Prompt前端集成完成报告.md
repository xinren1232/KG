## 🚀 使用指南

### 📍 访问路径
1. **打开前端**: http://localhost:5173
2. **导航路径**: 系统管理 → Prompt管理
3. **选择分类**: 抽取词典 (extraction)
4. **选择模板**: 词典抽取专家 或 词典抽取专家(详细版)

### 🎯 使用步骤

#### 第一步: 选择Prompt模板
```
简化版 (p003) - 适合:
├── 日常快速抽取
├── 熟练用户
└── 标准文档处理

详细版 (p004) - 适合:
├── 新手学习
├── 复杂文档处理
├── 团队培训
└── 质量要求高的场景
```

#### 第二步: 填写变量参数
- **document_content**: 粘贴待处理的技术文档内容
- **term_count_requirement**: 指定术语数量要求 (如"尽可能多"或"50个")
- **extraction_focus**: 指定抽取重点 (如"重点关注异常现象")

#### 第三步: 生成完整Prompt
- 点击"测试"或"生成"按钮
- 系统自动替换变量生成完整prompt
- 复制生成的prompt到AI助手中使用

#### 第四步: 执行抽取
- 将完整prompt提供给AI助手
- AI助手按照标准格式输出词典表格
- 检查输出质量并进行必要调整

## 📚 使用示例

### 📄 示例输入文档
```
手机摄像头模组在生产过程中常见的质量问题包括：
1. 对焦不准确 - 通常由于镜头组装偏差导致
2. 白平衡异常 - 可能是传感器校准问题
3. 噪点过多 - 与图像处理算法和传感器性能相关

检测工具包括：
- CCD视觉对位系统
- 色彩分析仪
- 分辨率测试卡

质量指标：
- 分辨率: 1200万像素
- 信噪比: >40dB
```

### 🎯 期望输出结果
| 术语 | 别名 | 类别 | 多标签 | 备注 |
|------|------|------|--------|------|
| 对焦不准确 | 对焦偏差;Focus Inaccuracy | Symptom | 影像相关;硬件相关;可靠性 | 镜头组装偏差导致的成像质量问题，影响拍照清晰度 |
| CCD视觉对位系统 | CCD Vision Alignment System | Tool | 工具;测试验证;影像相关 | 用于精确定位和检测的视觉系统，确保组装精度 |
| 分辨率 | Resolution;像素数 | Metric | 性能指标;影像相关;硬件相关 | 衡量图像清晰度的关键指标，单位为像素 |
